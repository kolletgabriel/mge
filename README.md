# Monitorias e Grupos de Estudo

**Para rodar o projeto:**
1. `cd front && npm install && npm run build`
2. `docker compose up -d`
3. `docker compose exec api python scripts/devseeds.py` (seeds extras de aluno e prof)

**Acesso:**
- app:
    - localhost:8000
    - credenciais do administrador:
        - user admin@admin.com
        - senha admin123
